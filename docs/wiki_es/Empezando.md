# Empezando con Carrera LTI

## ⚙️ Configuración Inicial
Para empezar a desarrollar o usar Carrera LTI, sigue estos pasos:

1.  **Clonar el repositorio**: `git clone <repo_url>`.
2.  **Instalar dependencias**: `npm install`.
3.  **Variables de Entorno**: Copia `.env.example` a `.env` y configura tus claves de Firebase y Google Gemini.
4.  **Iniciar**: `npm run dev`.

## 🗝️ Configuración de IA
Para usar las funciones de Aether y NexusAI, necesitas una clave de API de **Google AI Studio**.
- Configúrala en la sección de "Aether Chat" o en tu archivo `.env`.

## 📱 Instalación como App
Carrera LTI es una PWA. Puedes instalarla:
- **En Chrome**: Haz clic en el icono de instalar en la barra de direcciones.
- **En Móvil**: "Añadir a pantalla de inicio".
